比如spring mvc中，我们通常会实现的几个切面

  • 一个before advice的切面，实现输出请求日志

3. 小结

从上面的测试来看，不同的切面，默认顺序实际上是根据切面的命令来的；

  • A切面中的advice会优先B切面中同类型的advice
